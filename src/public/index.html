Food72 REST API Documentation

It is a very simple documentation for food72 API. Every request have root path http://arusremservis.ru/

Users

Get all users

GET /users

Get user by id

GET /users/userId
userId - The 12-byte ObjectId value. Comprises 24 characters, each has a value of [0-9 A-F a-f]

Get the user's likes or comments

GET /likes
GET /comments
headers: { "authorization": token }

Save new user

POST /users

body: {
"name": "userName",
"phone": "userPhone",
"password": "userPassword"
}
Will return new User and JWT-token

Authenticate user

POST /users/login
Will return user information and JWT-token

Dishes

Get all dishes

GET /dishes

Get dishes by id

GET /dishes/dishId
dishId - The 12-byte ObjectId value.

Get the likes|comments|toppings|locations|composition of dishes

GET /dishes/dishId/likes|comments|toppings|locations|composition

Add new dish

POST /dishes

Likes

Add like

POST /likes

Comments

Add comment

POST /comment

Toppings

Add topping

POST /topping

Locations

Add location

POST /locations
